软件上传指南

1

准备软件文件

确保您的软件文件完整且经过测试。支持的文件格式包括：

Windows: .exe, .msi, .zip macOS: .dmg, .pkg, .app.zip Linux: .deb, .rpm, .tar.gz, .AppImage Mobile: .apk, .ipa
2

克隆仓库到本地

使用 Git 将项目仓库克隆到您的本地计算机：

git clone https://github.com/zhang123233123123/softwaremanagement.git cd softwaremanagement
3

创建软件目录

在 software 目录下创建对应平台的文件夹结构：

software/ ├── windows/ │ └── your-software-name/ ├── macos/ │ └── your-software-name/ ├── linux/ │ └── your-software-name/ └── mobile/ └── your-software-name/
4

上传软件文件

将您的软件文件放入对应的目录中，建议文件大小不超过 100MB。如果文件较大，请提供下载链接。

5

更新软件数据

编辑 assets/js/main.js 文件中的 softwareData 数组，添加您的软件信息：

{ id: 9, // 新的ID name: "您的软件名称", description: "软件简短描述", platform: "windows", // windows/macos/linux/mobile icon: "fas fa-desktop", // FontAwesome图标 version: "1.0.0", size: "50.2 MB", downloads: 0, downloadUrl: "software/windows/your-software/file.exe", screenshots: [], category: "development", // 软件分类 tags: ["标签1", "标签2"], rating: 4.5, lastUpdated: "2024-01-15" }
6

提交更改

提交您的更改并推送到 GitHub：

git add . git commit -m "添加新软件: 您的软件名称" git push origin main
7

创建 Pull Request

如果您没有直接推送权限，请 Fork 项目并创建 Pull Request。我们会尽快审核您的提交。
